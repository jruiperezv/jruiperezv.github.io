1. Visual analytics dashboards for online learning

I have been the lead designer and developer in the implementation from scratch of two award-winning visual analytics dashboards:

1.1. ALAS-KA for Khan Academy

ALAS-KA includes more than 20 visualizations for Khan Academy with high level behavioural indicators such as for example hint abuse or video avoidance. ALAS-KA was implemented as part of my master thesis and has received several awards. You can read more about the different indicators and technical design in this paper about ALAS-KA, you can also watch a short demonstration video of ALAS-KA and check the source code which is open source in my GitHub account.

 

1.2. ANALYSE for Open edX

ANALYSE is another visual analytics dashboard but for Open edX environment. It contains 12 new visualizations regarding exercise, video and course activity that can be accessed by students and instructors. ANALYSE has been implemented as a project within the Telematics Departament at Universidad Carlos III of Madrid and has received several awards. You can read more information in this paper about the design of ANALYSE, you can also check this demonstration video of ANALYSE and see the open source code in my GitHub account.

 

2. Independent data science projects

I love dealing and making sense of data. I share here a couple of other independent data science projects I have developed:

2.1. Wikipedia word cloud generator

This is a fun project that takes as input a Wikipedia term, and web scrapes the HTML site to generate a word cloud of the English Wikipedia page for that term. You can use it online here.

 

2.2. Prediction of next word

This is a project to predict the most probable next word based on a ngram machine learning approach. The model has been trained using public stream data on the Internet (e.g. Twitter, web pages, blogs). It takes as input your text and it will update as you change the input. You can use it online here.

3. Research on Learning Analytics & Educational Data Mining

During my PhD research I applied data science techniques to online learning with emphasis on behavioral modelling, statistical inference and visual analytics. I share below a couple examples of international research studies:

3.1. Fraud and cheating in Massive Open Online Courses (MOOCs)

In this study we have developed an algorithm to detect students in MOOCs that cheat using multiple accounts to harvest solutions. We apply the algorithm on a MOOC from MIT, report the results and advice to decrease cheating. You can read the paper online.

 

3.2. Impact of Optional Activities in Virtual Learning Environments

In this study we analyze the impact of optional activities in online learning. We apply, among other things, categorical variable analysis and relationship mining of optional activities with learning outcomes. You can read the paper online.
